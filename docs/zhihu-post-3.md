### 一、STP 生成树协议动画（第 22 个）

STP 是很多人的盲区——背过了，但不知道它在网络里"动起来"是什么感觉。

现在有了 7 步完整动画：

1. 各交换机广播 BPDU
2. 根桥选举（Bridge ID 最小者胜）
3. 每台非根交换机确定根端口
4. 每条链路选出指定端口
5. 冗余端口进入阻塞状态（环路消除）
6. 拓扑收敛稳定
7. 链路故障后的快速收敛过程

考点里最容易混的"根端口/指定端口/阻塞端口"三种状态，动画里都有直观体现。

### 二、HCIA 考点对照表（新增工具）

备考 HCIA-Datacom 的同学应该都有一个痛点：**考纲里的考点很多，但不知道哪些有好的学习资源，哪些只能死背**。

我给网站加了一个「HCIA 考点地图」：

- 按 6 大模块组织（网络基础 / 以太网与交换 / 路由协议 / 传输层 / 应用层服务 / 网络安全）
- 每个考点标注状态：**✅ 有动画**（点击直接跳动画）/ **🔧 有工具** / **📖 理论**（只能背的那种）
- **可直接点击跳转**——看到"OSPF 链路状态路由 ✅"，直接点就进 OSPF 动画了，不用自己去找

目前覆盖：IP 路由、ARP、ICMP、VLAN、STP、OSPF、TCP（握手/挥手/拥塞）、UDP、DNS、DHCP、HTTP/HTTPS、NAT、TLS、FTP，共 **15+ 个考点**有配套动画。

---

### 三、子网划分计算器（新增工具）

子网划分是 HCIA 的必考手算题，但很多人对"为什么这样算"还是一知半解。

新的子网计算器不只给你答案，而是**把每一步计算过程都可视化**：

- 输入 IP 地址 + 前缀长度（/1 ~ /30）
- 看到 IP 的 **32 位二进制分解**，网络位和主机位用不同颜色区分
- 子网掩码的按位与运算过程清晰展示
- 直接输出：子网地址 / 广播地址 / 可用主机范围 / 主机数

对 VLSM 的理解帮助很大——"为什么 /25 只有 126 台主机"，看一眼二进制就明白了。
